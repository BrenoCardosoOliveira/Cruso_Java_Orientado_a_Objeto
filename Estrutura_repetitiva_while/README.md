# Exercícios – Estrutura Repetitiva While
## Exercício 01

### Descrição:
Escreva um programa que repita a leitura de uma senha até que ela seja válida.
Para cada senha incorreta, mostrar **"Senha Invalida"**.
Quando a senha correta (**2002**) for digitada, mostrar **"Acesso Permitido"** e encerrar.

**Exemplo**:
````
Entrada:
2200
1020
2022
2002

Saída:

Senha Invalida
Senha Invalida
Senha Invalida
Senha Invalida

 Acesso Permitido
````
## Exercício 02

### Descrição:
Leia coordenadas **(X, Y)** de vários pontos.
Para cada ponto, informe o quadrante:

 - primeiro

 - segundo

 - terceiro

 - quarto

O programa encerra quando **X = 0** ou** Y = 0, sem imprimir nada nessa linha final.

Exemplo**:
````
Entrada:
 2  2
 3 -2
-8 -1
-7  1
 0  2

Saída:
primeiro
quarto
terceiro
segundo
````
## Exercício 03

### Descrição:
Um posto deseja saber qual combustível é o preferido dos clientes.
Leia o código do combustível:

 - 1 → Álcool

 - 2 → Gasolina

 - 3 → Diesel

 - 4 → Fim

Se digitar um valor inválido, solicitar outro até vir um válido.
Ao digitar 4, finalizar e mostrar:

 - **MUITO OBRIGADO**

 - Quantidade de cada tipo abastecido.

**Exemplo**:
````
Entrada:
8
1
7
2
2
4

Saída:
MUITO OBRIGADO
Alcool: 1
Gasolina: 2
Diesel: 0
````
