## Exercício 06

Descrição:
Ler valores A, B e C (double) e calcular:

````
a) área do triângulo retângulo → (A * C) / 2
b) área do círculo → π * C²
c) área do trapézio → ((A + B) * C) / 2
d) área do quadrado → B²
e) área do retângulo → A * B
Considere π = 3.14159.
````
Exemplos:
````
Entrada:
3.0 4.0 5.2

Saída:
TRIANGULO: 7.800
CIRCULO: 84.949
TRAPEZIO: 18.200
QUADRADO: 16.000
RETANGULO: 12.000
````
````
Entrada:
12.7 10.4 15.2

Saída:
TRIANGULO: 96.520
CIRCULO: 725.833
TRAPEZIO: 175.560
QUADRADO: 108.160
RETANGULO: 132.080
````
